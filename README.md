# ☁️ Team Vibhav Cloud Computing Workshop Landing Page

Welcome to the official repository for the **Team Vibhav Cloud Computing Workshop** entry portal. This is an interactive, responsive web interface designed to capture participant registration data, perform client-side input validation, and generate dynamic event onboarding confirmations.

---

## 🚀 Features

* **Dynamic Text Generation:** Leverages JavaScript Template Literals to greet users natively using the exact name value provided during submission.
* **Smart Guardrails:** Features text-trim validation logic to prevent empty strings or empty form submittals.
* **Modern Utility Architecture:** Built with a modern, design-token approach to CSS utilizing custom properties (`:root`) for global color and font layouts.
* **Responsive Visual Framework:** Implements structural grid mappings and flex alignment to balance partner assets (`team vibhav logo.png` and `NImbusimg.png`) across screen breaks.

---

## 📂 Project Architecture

This project is fully self-contained across three foundational files:

```bash
